# 🧬 Breast Cancer Classification App

An interactive Machine Learning web application built with **Streamlit** and **Scikit-Learn** that predicts breast cancer diagnosis based on clinical features.

---

## 📌 Features

- **Interactive Web Interface:** User-friendly UI built with Streamlit.
- **Machine Learning Model:** Classification powered by Scikit-Learn.
- **Real-time Prediction:** Instant results based on input parameters.

---

## 🛠️ Tech Stack

- **Python** (Core Language)
- **Streamlit** (Web Application Framework)
- **Scikit-Learn** (Machine Learning Model & Data Pipeline)
- **Pandas & NumPy** (Data Processing & Manipulation)

---

## 🚀 Getting Started

Follow these instructions to get a local copy up and running on your machine.

### Prerequisites

Make sure you have Python 3.8+ installed on your system.

### Installation & Run
git clone https://github.com/mustafabashayer01-hub/Breast-Cancer.git
cd Breast-Cancer

Install the required dependencies:
Bash
pip install -r requirements.txt

Run the Streamlit app:
Bash
streamlit run breast_cancer.py
